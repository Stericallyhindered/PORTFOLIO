# BINFORGE

Modern BIN editing product shell built with Next.js and Supabase.

## Included

- Landing page with a premium positioning
- Username/password auth screen
- Vehicle-first garage dashboard
- Gorgeous BIN editor shell with table editing, 3D surface graph, and AI sidecar toggle
- Supabase helpers and starter schema
