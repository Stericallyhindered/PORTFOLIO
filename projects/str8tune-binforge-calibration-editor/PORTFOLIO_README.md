# str8tune-binforge-calibration-editor

Next.js ECU/BIN editing product shell for calibration-style workflows. This project shows frontend/product work around binary editing concepts, table/map views, tuning-oriented UI, Supabase-backed structure, and modern TypeScript app organization.

## What It Shows

- Next.js/TypeScript product shell for calibration/BIN editing workflows.
- App, component, and library organization for tuner-facing tools.
- Supabase-backed project structure.
- A domain-specific software tool rather than a generic CRUD demo.

## Key Files

- `app` - Next.js route/app structure.
- `components` - UI components for the product shell.
- `lib` - shared app logic.
- `supabase` - backend/database structure.
- `package.json` - stack and dependencies.
