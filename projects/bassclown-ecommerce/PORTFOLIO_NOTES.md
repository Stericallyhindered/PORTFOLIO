# Bass Clown Ecommerce

Public portfolio snapshot of a polished Next.js ecommerce/customer workflow project.

## What To Review

- `app/` - Next.js route structure and authenticated/customer-facing workflows.
- `components/` - reusable UI and admin/customer components.
- `lib/` - backend helpers, integrations, and shared logic.
- `drizzle/` and `migrations/` - database schema/migration work.
- `database-setup.sql` and `restore-database-schema.sql` - database structure examples without private production data.
- `API_DOCUMENTATION.md`, `PROJECT_STATUS.md`, `COMPLETION_REPORT.md` - implementation and project-status documentation.

## Why It Matters

This shows product polish and business delivery: ecommerce UI, auth, database workflows, media/product management, admin surfaces, handoff documentation, and the kind of full-stack detail clients actually need.

## Public Snapshot Notes

Production data restore files, admin seed scripts, workflows containing secret names, local setup clutter, and generated/test directories were removed before publishing.
