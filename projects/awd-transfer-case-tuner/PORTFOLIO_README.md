# awd-transfer-case-tuner

Flutter AWD tuner project for vehicle drivetrain/control experimentation. This was copied only from the actual `awd_tuner` project folder, not the unsafe parent folder, and shows the mobile/software side of transfer-case or AWD tuning workflows.

## What It Shows

- Flutter app structure for a vehicle-control/tuning tool.
- Android build configuration and app source appropriate for mobile diagnostics/configuration workflows.
- A focused vehicle software example separate from unrelated files in the parent directory.

## Key Files

- `lib` - Flutter application source.
- `pubspec.yaml` - app dependencies and platform setup.
- `test` - available app tests.
- `README.md` - original project notes.
