# Stealth Machine Tools Flutter App

Public portfolio snapshot of a cross-platform Flutter app for Stealth Machine Tools support, training, and field-service workflows.

## What To Review

- `lib/` - Flutter application source, screens, services, models, providers, and core config.
- `lib/core/config/dev_config.dart` - portfolio-safe configuration structure with real credentials removed.
- `pubspec.yaml` - app dependencies for state management, API calls, local storage, auth helpers, speech, camera, PDF/document viewing, QR, charts, and routing.
- `test/` - Flutter test structure.

## Why It Matters

This shows the same class of mobile work as the hardware apps: cross-platform Flutter, support workflows, authenticated API integration, local persistence, training/support material UX, and app structure intended for field users rather than a toy demo.

## Public Snapshot Notes

Large manuals, contracts, generated Flutter ephemeral folders, dependency output, and credentials were removed before publishing.
