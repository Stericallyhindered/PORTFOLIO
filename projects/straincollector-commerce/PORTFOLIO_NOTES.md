# StrainCollector Commerce

Public portfolio snapshot of a specialty ecommerce/order workflow project built with Next.js, Supabase, TypeScript, and supporting PHP scripts.

## What To Review

- `src/` - Next.js application source.
- `CURRENTORDERTRACKING/` - order tracking workflow work.
- `supabase/` - database/backend project structure.
- `scripts/` - project automation scripts.
- `create-checkout-session.php`, `stripe-webhook.php`, `create-shipping-label.php`, `trackingslips.php` - payment, webhook, shipping, and tracking workflow examples.
- `README.md`, `CHECKOUT_FLOW.md`, `ORDER_MANAGEMENT_GUIDE.md` - workflow documentation.

## Why It Matters

This shows practical commerce operations: checkout flow, order management, labels/tracking, customer-facing pages, backend glue code, and deployment/support notes.

## Public Snapshot Notes

Secrets, config with live values, test endpoints, large product images, scraped/generated assets, and private deployment notes were removed before publishing.
