# LaserConsumables Commerce

Public portfolio snapshot of a Next.js/TypeScript commerce/admin system for industrial laser consumables.

## What To Review

- `app/` - app routes, admin pages, API flows, and middleware.
- `components/admin/ShippingDashboard.tsx` - shipping/admin workflow UI.
- `components/admin/ShipStationDashboard.tsx` - ShipStation-style operations dashboard.
- `components/admin/ManualOrderForm.tsx` - manual order creation workflow.
- `lib/services/inventory-advanced.ts` - inventory service logic.
- `prisma/schema.prisma` - commerce data model.

## Why It Matters

This shows full-stack business software around catalog/admin workflows, orders, inventory, shipping operations, authentication, Prisma data modeling, and production-style Next.js structure.

## Public Snapshot Notes

Generated storefront HTML, scraped product pages, deployment docs, workflow secret references, build output, and private credentials were removed before publishing.
