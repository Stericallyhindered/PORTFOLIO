# Stealth Batteries AI Support System

This is an early customer-support system built around the Stealth Batteries support workflow: customer chat, admin review, conversation logging, ticket creation, escalation, and Claude-powered support responses.

The goal was simple: reduce manual support load while still keeping the workflow reviewable and useful for the business. Customers could ask questions through chat, the system could respond with AI-assisted support, and harder conversations could be flagged for human follow-up instead of getting lost in email.

## What It Shows

- Claude/Anthropic API integration through environment configuration
- Real-time chat/support flow concepts
- Admin dashboard and conversation review patterns
- Support ticket creation and escalation handling
- SQLite-style persistence for conversations and support data
- Practical customer-support automation, not just a chatbot demo

## Why It Matters

This project fits the kind of work I like doing: taking an overloaded manual process and turning it into a tool that gives the business structure, visibility, and faster response times.

In a production version, the important parts are not only the model response. The surrounding system matters more: customer identity, conversation state, escalation rules, support history, admin visibility, logging, and clear failure handling.
