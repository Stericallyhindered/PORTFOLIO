# ESP32 Mouse Controller

Small embedded/desktop control project for an ESP32-based USB HID mouse controller.

## What To Review

- `esp32_mouse_controller.ino` - ESP32/Arduino firmware for USB HID mouse behavior.
- `esp32_mouse_test.ino` - test firmware scaffold.
- `mouse_controller_gui.py` - Python GUI for controlling and testing behavior from the desktop.
- `mouse.py` - supporting Python control logic.
- `requirements.txt` - Python dependencies.

## Why It Matters

This is a compact example of hardware-connected software: firmware, USB HID behavior, serial/device control, and a desktop GUI used to configure/test a physical device.
